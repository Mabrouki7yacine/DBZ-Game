# DBZ-Game
A simple 2D game prototype built in **C++** using **Raylib** and **CMake**.
---

## Project Structure
├── main.cpp # Entry point
├── src/Person.cpp # Character logic
└── Include/ # Headers (.h)
└── Resources/ # Images, audio, fonts, etc.

Build & Run (Linux)
```bash
cmake . build/
cmake --build build/
./build/game_v1
